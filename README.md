Moodspace - веб-приложение для получения персональных рекоммендаций контента и трекинга эмоционального состояния.

Пользователь может выбрать своё настроение на данный момент, получить подборку фильмов, сериалов, книг и игр, вести дневник воспоминаний

Возможности:

- регистрация и авторизация пользователей;
- выбор настроения;
- персональные рекомендации;
- дневник воспоминаний с загрузкой изображений;
- добавление рекомендаций в избранное;
- отметки "просмотрено" и "не понравилось";
- профиль пользователя со статистикой;
- REST API для получения рекомендаций.

Технологии:

- Python
- Django
- Django REST Framework
- Bootstrap 5
- SQLite
- HTML/CSS
